- Typescript

Without any UI libraries

## Getting Started

To run the project, firstly

### `npm install`

To install required dependencies

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


### `npm test`

Launches the tests
